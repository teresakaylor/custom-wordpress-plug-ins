## üîó How to Use

1.  **Clone the Repository:** Clone this repository to your local machine using the following command:
    ```bash
    git clone [https://github.com/teresakaylor/custom-wordpress-plugins.git](https://github.com/teresakaylor/custom-wordpress-plugins.git)
    ```
2.  **Select Plugin:** Navigate into the subdirectory of the plugin you wish to use (e.g., `spot-wordpress-wheel-plugin`).
3.  **Installation:** Copy the chosen plugin folder into your WordPress installation's `/wp-content/plugins/` directory.
4.  **Activate:** Activate the plugin through the WordPress admin dashboard.
